# Summon Night: Swordcraft Story PT-BR

Projeto de tradução para Português Brasileiro de **Summon Night: Swordcraft Story** (Game Boy Advance).

## Objetivos

- Traduzir o jogo para PT-BR.
- Desenvolver ferramentas para auxiliar a tradução.
- Documentar a estrutura da ROM.

## Tecnologias

- Python
- Linux
- mGBA
- GHex

## Estrutura

- `scripts/` - Ferramentas desenvolvidas para o projeto.
- `translation/` - Arquivos de tradução.
- `fonts/` - Modificações na fonte do jogo.
- `patches/` - Patches distribuídos.
- `docs/` - Documentação técnica.
